¿Qué es?

OwnCloud es una aplicación de software libre del Servicio de alojamiento de archivos que permite el almacenamiento en línea y aplicaciones en línea (cloud computing).OwnCloud puede ser instalado dentro de un servidor que disponga de una versión reciente de PHP (mayor o igual a 5.6) y soporte de SQLite (base de datos por defecto), MySQL o PostgreSQL.

Instalación

  • CREAMOS UNA BASE DE DATOS

    Ejecutamos el siguiente comentario (nos pedirá la contraseña root del MySQL):

    mysql -uroot -p

    La creamos:
    CREATE DATABASE owncloud;
    CREATE USER admin@localhost IDENTIFIED BY 'contraseña';
    GRANT ALL PRIVILEGES ON owncloud.* TO admin@localhost;
    FLUSH PRIVILEGES;
    exit

  • DESCARGAMOS OWNCLOUD

    Vamos a descargar los archivos de owncloud de su sitio web.

    cd ~
    wget http://download.owncloud.org/community/owncloud-latest.tar.bz2

    Con esto habremos descargado el archivo comprimido con el contenido de owncloud en nuestra carpeta raíz. Podemos extraer los archivos para reconstruir el directorio de owncloud escribiendo :

    tar tar -xjf owncloud-latest.tar.bz2

    tar sudo mv owncloud /var/www/html/

    Esto creará un directorio llamado owncloud en nuestra carpeta raíz.
    Despues hacemos un update.

    sudo apt-get update

    Tras ello, deberemos de obtener los dos paquetes necesarios :

    apt-get install php libapache2-mod-php php-mcrypt php-mysql

    Despues deberemos cambiar el nombre del DirectoryIndex para que coja el index.php

    nano /etc/apache2/mods-enabled/dir.conf

    IfModule mod_dir.c DirectoryIndex index.html index.cgi index.pl index.php index.xhtml index.htm /IfModule>
    IfModule mod_dir.c> DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm /IfModule>
