¿Qué es?

WordPress es un sistema de gestión de contenidos que nos permite crear una página web y modificarla a nuestro antojo, metiéndole toda clase de plugins y la información que nos apetezca. Es software libre pero puede llegar un punto en el que si se usan demasiado plugins o se usan demasiadas utilidades en la página podrían solicitar cobrarnos por ello.

Instalación

  • CREAMOS UNA BASE DE DATOS

    Ejecutamos el siguiente comentario (nos pedirá la contraseña root del MySQL):

    mysql -uroot -p

    La creamos:
    CREATE DATABASE wordpress;
    CREATE USER wordpressuser@localhost IDENTIFIED BY 'contraseña';
    GRANT ALL PRIVILEGES ON wordpress.* TO wordpressuser@localhost;
    FLUSH PRIVILEGES;
    exit

  • DESCARGAMOS WORDPRESS

    Vamos a descargar los archivos de WordPress de su sitio web.

    cd ~
    wget http://wordpress.org/latest.tar.gz

    Con esto habremos descargado el archivo comprimido con el contenido de WordPress en nuestra carpeta raíz. Podemos extraer los archivos para reconstruir el directorio de WordPress escribiendo :

    tar xzvf latest.tar.gz

    Esto creará un directorio llamado Wordpress en nuestra carpeta raíz.
    Despues hacemos un update.

    sudo apt-get update

    Tras ello, deberemos de obtener los dos paquetes necesarios :

    sudo apt-get install php5-gd libssh2-php

    Esto permitirá que trabajemos con imágenes, que instalemos plugins.

  • CONFIGURAMOS EL WORDPRESS

    Empezamos moviéndonos hacia el directorio de WordPress que acabamos de descomprimir:

    cd ~/wordpress

    Un archivo de configuración de ejemplo , que se acerca mucho al que necesitamos, está incluido por defecto. Aunque, necesitamos copiarlo a la localización del archivo de configuración por defecto para que WordPress reconozca al archivo. Lo haremos así:

    cp wp-config-sample.php wp-config.php

    Ahora que tenemos un archivo de configuración con el que trabajar, lo abriremos con un editor de texto :

    nano wp-config-php

    Necesitamos encontrar las configuraciones de DB_NAME , DB_USER y DB_PASSWORD para que WordPress pueda conectar y autenticarse correctamente en la base de datos creada .
    Habria que corregir los parametros y poner los valores que anteriormente pusimos.

    Ahora que tenemos nuestra aplicación configurada, necesitamos copiarla dentro de la raíz de documentos de nuestro servidor web, desde donde podrá ser servida a los visitantes de nuestra web.
    Copiamos los archivos en la raíz de documentos en el LAMP de Ubuntu (/var/www/html):

    sudo cp -r ~/wordpress/* /var/www/html/

    Ahora nos movemos hacia dentro de la raíz de documentos para hacer unos cambios finales a los permisos:

    cd /var/www/html

    Le damos los permisos:

    sudo chown –R nombreroot:www-data *

    Después creamos el que será el directorio padre de nuestro contenido:

    mkdir /var/www/html/wp-content/uploads

    Le damos los permisos:

    sudo chown -R :www-data /var/www/html/wp-content/uploads

    Esto permitirá al servidor web crear archivos y directorios a partir de este mismo directorio, lo cual nos permitirá subir contenido al servidor.

Uso básico

  • MENU

    Este es el menu del wordpress en el que podemos editar todo lo que queramos de nuestra pagina web.Podemos meter nuevos post,videos ,hacer paginas nuevas ,añadir comentarios ,cambiar la apariencia,crear usuarios y añadir plugins.

  • APARTADO "PLUGINS"

    En el apartado de plugin podemos instalar,buscar,desisntalar plugins. Hay muchísimos plugins disponibles ya que la gente crea y sube los suyos.
    Teniendo eso en cuenta, también puede significar que algunos de esos plugins no funciones correctamente.

  • APARTADO "POSTS"

    En este apartado podemos ver los distintos post que han sido subidos a nuestro WordPress y clasificarlos por categoría, fecha... etc.
    Aparte, podemos crear posts nuevos y eliminar los ya existentes.

  • APARTADO "MEDIA"

    Es el apartado que sirve para almacenar todos los audios,videos e imágenes que metamos dentro de nuestro WordPress de forma ordenada. También se pueden añadir más audios, vídeos o imágenes desde ahí.
